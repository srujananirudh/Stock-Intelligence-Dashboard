# Stock-Intelligence-Dashboard

A dynamic and interactive stock analytics dashboard built using FastAPI, Pandas, and JavaScript.
This project provides financial insights and visualizations for companies based on real-world datasets.

---

## 🚀 Features

* 📌 Company-wise filtering
* 📊 Key financial metrics:

  * EPS (Earnings Per Share)
  * Net Profit
  * Revenue
  * Operating Profit Margin
* 📈 Interactive charts (Trend Analysis)
* 🔄 Live API-based data fetching
* 🎯 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Backend:** FastAPI (Python)
* **Data Processing:** Pandas
* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** Chart.js

---

---

## ⚙️ How to Run

### 1️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 2️⃣ Run backend

```
uvicorn main:app --reload
```

### 3️⃣ Open dashboard

* Open `index.html` using Live Server (VS Code)
